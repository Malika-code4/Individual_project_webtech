Memory Master - web App

An interactive Memory Card Game web application with a beautiful, full of features, 3D game board, various levels of difficulty, real-time statistics, and world leaderboard.

Dynamic Gameplay
  - Solo Mode: Challenge yourself to beat your best time and score.
  - Difficulty Levels: Choose from Easy (4x4), Medium (6x4), and Hard (6x6) grids.
  - 3D Card Effects: Smooth flip animations and visual feedback.

Player Progression
  - Profile System: Select your own username and add your own avatar.
  - Statistics: Keep a record of the total number of games played, the number of wins, average moves and highest points
  - Leaderboard: Take on the highest spot in the world, sortable by difficulty.


Modern UI/UX
  - Responsive Design: Optimized to the full extent both on desktop and mobile.
  - Heads-Up Display (HUD): In-game tracking of time and moves found in pairs.
  - Glassmorphism Aesthetic: Smooth, current interface using dark mode.

Technology Stack


Frontend: HTML, CSS, JavaScript.
Backend: PHP 8 (Object-Oriented Architecture)
- Database: MySQL 
Database: MySQL
Architecture: API communication based on REST.
